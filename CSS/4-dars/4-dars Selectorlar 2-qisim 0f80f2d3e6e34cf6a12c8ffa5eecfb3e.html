4-dars. Selectorlar 2-qisim.

[disabled] {
    background-color: blueviolet;
}
[id] {
    border: 3px solid green;
    margin: 10px;
}
type = "text" bo’lgan barcha atributlarni tanlab olish. 
[type = "text"] {
    border: 2px solid blue;
    margin-bottom: 10px;
    outline: none;
}
type atributi qiymati “te” bilan boshlangan, barcha atributlarni tanlab olish 
[type^="te"] {
    background: green;
}
type atributining qiymati “text” tugagan barcha atributlarni tanlab olish. 
[type$="text"] {
    background: green;
    border: 5px double red;
    margin-bottom: 10px;
}
atributning qiymatining matnining ixtiyoriy qismidan tanlab olish 
[type*="ex"] {
    background: green;
    border: 5px double red;
    margin-bottom: 10px;
}

Filtirlash

Ham qizillar sinfiga mansub, ham paragrif bo’lganlarini tanlab olish. 
p.qizillar {
    color: red;
}
<sel1><sel2>...<selN>

Elementlarni uning otasiga ko’ra tanlab olish. 
<sel1> > <sel2> > ... > <selN>
Shunday paragriflarni tanlab olinki uning otasi id html bo’lsin. 
#html > p {
    color: green;
}
Elementlarni ularning ajdodiga ko’ra tanlab olish. 
<sel1> <sel2> ... <selN>
Shunday a elementlarini tanlab olingki, ajdodi nimadir sinfiga mansub bo’lsin. 
.nimadir a {
    text-decoration: none;
    color: inherit;
}
Shunday litle nomli classga ega spanning ajdodi a va uning ajdodi nimadir classiga mansublarini tanlab olish 
.nimadir a span.litle{
    border: 2px solid green;
}
Elementlarni ularning akasiga ko’ra tanlab olish 
<sel1> ~ <sel2> ~ ... ~ <selN>
Shunday paragriflarni tanlab olingki, akalarini orasida h2 bo’lsin. 
h2 ~ p {
    color: blue;
}
Shunday span nomli elementni tanlab olingki, uning otasi a va uning akasi p va uning ajdodi div bo’lsin. 
div p ~ a > span {
    color: blue;
}
Elementlarni ularning eng kichik akasiga ko’ra tanlab olish. 
<sel1> + <sel2> + ... + <selN>
Shunday paragrifni tanlab olki, uning eng kichik akasi h2 bo’lsin. 
h2 + p {
    color: blue;
}
:checked - check qilinganlarni tanlab olish. 
input:checked + span {
    color: blue;
}
